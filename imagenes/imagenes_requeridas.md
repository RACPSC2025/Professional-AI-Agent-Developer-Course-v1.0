## Fuentes Recomendadas

Para obtener imágenes de alta calidad, considere estos recursos:

1. **Icons8** (https://icons8.com/) - Amplia colección de iconos
2. **Flaticon** (https://www.flaticon.com/) - Miles de iconos gratuitos
3. **Heroicons** (https://heroicons.com/) - Iconos minimalistas
4. **Feather Icons** (https://feathericons.com/) - Iconos limpios y simples
5. **Simple Icons** (https://simpleicons.org/) - Iconos de marcas y tecnologías
6. **Unsplash** (https://unsplash.com/) - Imágenes de alta calidad gratuitas

## Requisitos Técnicos

- **Formato**: PNG para transparencia, SVG para escalabilidad
- **Tamaño**: 16x16 a 32x32 px para iconos de texto, 200x200 px para imágenes de sección
- **Estilo**: Estilo plano o lineal para mantener consistencia visual
- **Color**: Paleta pastel para mantener la estética minimalista del curso
