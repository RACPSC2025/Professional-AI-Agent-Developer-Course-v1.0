# Microsoft AI Agents for Beginners

## Complete Course: 12 Lessons to Start Building AI Agents

### Course Description

This complete course is designed to introduce you to the world of artificial intelligence agents developed with Microsoft technologies. Throughout 12 lessons, you'll learn from basic concepts to advanced applications in production environments.

### Main Lessons (12 Lessons):

1. **Intro to AI Agents and Agent Use Cases**
   - Introduction to AI agents and their practical applications
   - Business use cases and real-world examples
   - Agent architecture fundamentals

2. **Exploring AI Agentic Frameworks**
   - Overview of the main AI agent frameworks
   - Comparison between Microsoft Agent Framework, AutoGen, Semantic Kernel
   - Selecting the right framework for your use case

3. **Understanding AI Agentic Design Patterns**
   - Key patterns for building effective agents
   - Designing scalable and maintainable architectures
   - Best practices for agent development

4. **Tool Use Design Pattern**
   - How agents can utilize external tools
   - Integration with APIs, databases, and business systems
   - Secure tool management for external systems

5. **Agentic RAG (Retrieval-Augmented Generation)**
   - Retrieval-Augmented Generation for agents
   - Context management and memory
   - Practical examples of RAG in agents

6. **Building Trustworthy AI Agents**
   - Principles for creating reliable and secure agents
   - Security and privacy considerations
   - Testing and validation of agents

7. **Planning Design Pattern**
   - Agents that can plan and execute complex tasks
   - Hierarchical and sequential planning
   - Resource dependency and management

8. **Multi-Agent Design Pattern**
   - Working with multiple agents collaborating
   - Agent communication and task coordination
   - Multi-agent system architectures

9. **Metacognition Design Pattern**
   - Agents that think about their own thinking
   - Self-reflection and self-assessment capabilities
   - Iterative performance improvement of agents

10. **AI Agents in Production**
    - Deployment and management of agents in real-world applications
    - Scalability and performance considerations
    - Monitoring and maintenance of production agents

11. **Using Agentic Protocols (MCP, A2A and NLWeb)**
    - Communication between agents using specific protocols
    - Integration with enterprise systems using MCP
    - Standards and best practices for agent communication

12. **Context Engineering for AI Agents**
    - Context management and creation for agents
    - State persistence and session continuity
    - Contextual adaptation based on environment

### Technologies Covered:
- Microsoft Agent Framework (MAF)
- Azure AI Agent Service
- Semantic Kernel
- AutoGen (transitioning to MAF)
- Model Context Protocol (MCP)
- Azure OpenAI Service
- Azure Cognitive Services

### Prerequisites:
- Basic programming knowledge (preferably Python or C#)
- Experience with REST APIs
- Azure account for AI services
- Understanding of basic AI and language model concepts

### Target Audience:
- Developers interested in creating AI agents
- Software engineers who want to integrate AI into their applications
- Solution architects planning AI implementations
- Technical professionals who need to understand AI agent capabilities

This course provides a progressive learning path from fundamental concepts to production applications, covering both theory and practical implementation of AI agents with Microsoft technologies.
